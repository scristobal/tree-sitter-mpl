# tree-sitter-mpl

Tree-sitter grammar and queries for Axiom Metrics Processing Language (MPL).

Personal use, but try out on the examples

```sh
tree-sitter highlight --grammar-path . --scope source.mpl examples/*.mpl
```

or build with

```sh
tree-sitter generate && tree-sitter build
```
